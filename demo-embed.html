This is a demo page showing how the ShieldBase chatbot widget can be embedded on any website.

Embed Instructions

Add one script tag before the closing </body> tag of your website:

<script src="http://localhost:5173/embed.js" data-shieldbase data-url="http://localhost:5173"></script>

Options

Customize the widget with data attributes:

<script src="http://localhost:5173/embed.js" data-shieldbase data-url="http://localhost:5173" data-position="bottom-right" data-color="#1e3a5f" data-title="ShieldBase Chat" ></script>

data-url — URL where the chatbot is hosted (default: http://localhost:5173)

data-position — bottom-right, bottom-left, top-right, or top-left

data-color — Primary color for the button and header (hex)

data-title — Title shown in the widget header

Try It

Click the chat bubble in the bottom-right corner to open the widget!
